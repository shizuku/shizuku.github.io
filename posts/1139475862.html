基础知识

git 是一种分布式的版本控制系统，由 linus 大神开发，最初是用来管理 Linux 的代码。

git 采用分支和

基本操作

创建仓库

在 shell 中输入：

git init

可以将当前目录初始化成一个 git 仓库

克隆仓库

使用：

git clone https://path/to/repo

可以克隆链接所指的仓库到当前目录下。链接所指的可以是本地的仓库也可以是远程的仓库。

添加文件

使用

git add

添加一个文件的更改。使用下面几个命令中的一个添加当前目录下所有的更改。

git add .
git add *
git add --all
